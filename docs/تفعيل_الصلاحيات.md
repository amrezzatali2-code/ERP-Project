# تفعيل الصلاحيات في البرنامج

## الخدمة الجاهزة

تم إضافة `IPermissionService` و `PermissionService` للتحقق من صلاحيات المستخدم.

### الاستخدام في Controller

```csharp
public class SalesInvoicesController : Controller
{
    private readonly IPermissionService _permService;

    public SalesInvoicesController(IPermissionService permService)
    {
        _permService = permService;
    }

    public async Task<IActionResult> Create()
    {
        if (!await _permService.HasPermissionAsync(ERP.Security.PermissionCodes.SalesInvoices.Create))
        {
            return RedirectToAction("AccessDenied", "Home");
        }
        // ... باقي الكود
    }
}
```

### الاستخدام في View

```html
@inject ERP.Services.IPermissionService PermService

@if (await PermService.HasPermissionAsync(ERP.Security.PermissionCodes.SalesInvoices.Edit))
{
    <button>تعديل</button>
}
```

### أكواد الصلاحيات (PermissionCodes)

راجع `Security/PermissionCodes.cs` لقائمة الصلاحيات، مثل:
- `SalesInvoices.Create`, `SalesInvoices.Edit`, `SalesInvoices.Delete`
- `Products.View`, `Products.Create`, `Products.Edit`
- `Accounts.CashReceipt`, `Accounts.CashPayment`
- وغيرها...

## آلية العمل

الصلاحية النهائية للمستخدم = (من أدواره) + (صلاحيات إضافية) - (صلاحيات ممنوعة)
